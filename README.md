# 💻 Clube do livro BookWorm

Este é um projeto de site para a avaliação de Front-End, que avalia a capacidade de desenvolver interatividade em sistemas utilizando JavaScript. Este sistema é de um site do clube do livro BookWorm, que vende livros.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Estilização e Feedback Visual)
* JavaScript (Manipulação de DOM e Eventos)

## 📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado:
/  
├── index.html  
└── src/  
        ├── styles/  (Estilos da página)  
        ├── scripts/ (Lógica de interação)
        └── images/  (Ativos visuais)  

## 🚀 Funcionalidades Implementadas
* **Interatividade Visual:** O banner principal do site muda a suas cores e seu tamanho quando o usuário passa o mouse sobre ele.
* **Cálculo do Preço Total da Compra:** Formulário processa a entrada do usuário e automaticamente calcula e exibe o valor total da compra, já considerando o preço promocional de R$42,00.
* **Lista de Desejos:** Formulário que adiciona, quando um botão é clicado, as entradas do usuário à uma lista com cards dos itens já adicionados, com verificação de erro caso uma entrada vazia tente ser adicionada.
* **Limpar a Lista de Desejos:** Remoção de todos os itens adicionados pelo usuário ao clicar no botão, junto de um aviso sobre a ação que acaba de ser feita e do foco ao campo de texto para recomeçar a adicionar itens na lista.
